6 大核心功能

从监控到还原，全程覆盖

核心
⚡
毫秒级文件追踪 · 自动存档

基于 macOS FSEvents 系统级监控，在完全无感知的情况下，捕捉每一次文件的创建、修改、删除。内容未实际变更则不记录。

使用 APFS clonefile 技术，备份极速，几乎不占额外磁盘空间
文件内容哈希比对，打开但未修改不会产生记录
存档元数据存入本地 SQLite，内容通过内容寻址存储
可按目录配置监控范围，支持多层子目录过滤
智能
◎
AI 任务独立记录

当 Cursor、Claude Code 等 AI 工具运行时，该时段的文件变更自动归入 AI 任务记录，与定时存档分开展示。

每次 AI 任务完成后生成独立存档条目
AI 任务期间定时存档自动暂停，避免重复
支持 Cursor / Claude Code / Codebuddy / Windsurf 等工具
任务名称、工具来源、文件数量一目了然
核心
⇄
文件级 diff 对比

点击任意存档记录，即可在三栏布局中查看每个文件的前后变更。新增、删除、修改，一目了然。

三栏布局：时间线 / 文件列表 / diff 详情
高亮显示新增、删除、修改行
支持大文件快速 diff，性能经过优化
从 diff 视图直接发起单文件读档
核心
↩
一键读档 · 任意历史节点

像游戏读档一样回到任意历史状态。支持整个存档点整体还原，也支持精确到单个文件。

整体读档：将存档点内所有文件回到历史状态
单文件读档：只还原选中的一个文件
读档状态持久化存储，刷新后仍可感知
两步确认机制防止误操作
安全
⊘
异常熔断告警

检测到短时间内大量文件被删除或修改时，立即发出告警通知，让你在 SSD TRIM 执行前介入。

可配置熔断阈值（文件数量 / 时间窗口）
macOS 原生通知，第一时间告知异常
AI 越界操作场景下的最后一道防线
正常 AI 任务不会误触发
配置
⚙
目录级精细配置

每个监控目录独立配置忽略规则。存档时间周期可调，从 1 分钟到 2 小时。

支持任意层级子目录和单文件排除
更改存档周期后，当前存档立即封存
可手动触发重新扫描
所有配置本地存储，重启后自动恢复
三步上手

两分钟，开始保护你的文件

不需要命令行，打开 App 按步骤操作即可。

rew — 上手流程 macOS 可用
01
选择要保护的目录

打开 rew，在设置面板点击「添加目录」，选择你的工作目录。rew 会立即开始扫描并建立初始存档基线。

02
后台静默保护

rew 在菜单栏静默运行。每当 AI 工具或任何程序修改文件，都会自动记录快照。

03
出问题？一键读档

在时间线中找到目标存档点，点击「读档」回到历史状态。

$ rew undo
