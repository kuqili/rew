rew CLI & Daemon

Designed for focus.

Let AI write the code. We'll make sure you never lose it.

Automatic Snapshots

Zero configuration required. rew leverages APFS clonefile technology to instantly backup your files the moment an AI tool touches them, with practically zero storage overhead.

Anomaly Detection

If Claude or Cursor goes rogue and deletes or modifies dozens of files in seconds, rew immediately alerts you and locks the last safe snapshot.

Native Performance

Written in Rust. Fully local. rew uses macOS FSEvents to monitor changes without polling, ensuring your CPU and battery remain completely untouched.
